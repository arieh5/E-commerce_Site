Ce projet a été réalisé en binôme dans le cadre de ma formation à l'Université Paris 1 Panthéon-Sorbonne. L'objectif était de concevoir et de développer un site de vente en ligne entièrement fonctionnel, intégrant une gestion dynamique des données.

Fonctionnalités techniques
Gestion de base de données : Création et administration d'une base de données relationnelle via MySQL pour stocker les produits, les utilisateurs et les commandes.

Développement Backend : Implémentation de la logique métier en PHP (système de panier, authentification, filtrage des produits).

Interface Frontend : Conception d'une interface utilisateur ergonomique en HTML/CSS.

Travail collaboratif : Utilisation de méthodes de développement en équipe pour assurer la cohérence du code et des fonctionnalités.
